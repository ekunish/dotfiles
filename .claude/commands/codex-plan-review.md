Review the latest plan file using OpenAI Codex CLI.

## Steps

1. Find the most recently modified plan file in `~/.claude/plans/`
2. Run the review via Bash:
   ```
   bash ~/.claude/codex-plan-review.sh
   ```
3. Display the Codex review results to the user
